# 🛒 Sistema de Cadastro e Análise de Produtos

Sistema desenvolvido em Python para registrar produtos e analisar informações de vendas, como atividade avaliativa da disciplina **Programming & Algorithms** — Faculdade Impacta Tecnologia.

---

## 📋 Funcionalidades

- ✅ Cadastro de 5 produtos com nome e preço
- ✅ Validação de entradas numéricas com tratamento de erros
- ✅ Cálculo do valor total dos produtos
- ✅ Identificação do produto mais caro e mais barato
- ✅ Cálculo da média de preços
- ✅ Busca de produto pelo nome

---

## 🚀 Como utilizar

1. Acesse o notebook pelo link do Google Colab acima
2. Execute a célula com o código
3. Siga as instruções exibidas no terminal

> Desenvolvido e testado no Google Colab. Não requer instalação de dependências externas.

---

## 🗂️ Estrutura do código

| Função | Descrição |
|---|---|
| `calcular_total()` | Retorna a soma de todos os preços |
| `produto_mais_caro()` | Retorna o produto com maior preço |
| `produto_mais_barato()` | Retorna o produto com menor preço |
| `exibir_produto()` | Busca um produto pelo nome |
| `main()` | Função principal do sistema |

---

## 🛠️ Tecnologias

- Python 3
- Estruturas utilizadas: dicionários, funções, loops `for` e `while`, tratamento de exceções com `try/except`
